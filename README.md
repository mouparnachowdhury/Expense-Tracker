# Expense Tracker

My second project while learning to code.

After I built a small to-do tracker, I wanted to try something with a bit more
going on. I kept losing track of small daily spending, so I made this to add up
what I spend and show me where the money actually goes.

You type what you spent, how much, and pick a category. The app keeps a running
total and draws little bars showing which categories eat up the most money.

## What it does

- Add an expense with a description, amount, and category
- See your total spending at the top
- See a breakdown by category, with bars and percentages
- Filter the list to one category
- Delete anything you added by mistake
- Remembers everything even after you close the tab

## Built with

- HTML
- CSS
- JavaScript

No frameworks or libraries. I wanted to do the math and the bars myself so I
understood how it works.

## How to run it

1. Download or clone this repository.
2. Open `index.html` in any web browser.

No installation needed.

## What I learned building this

- Grouping data by category and adding up totals
- Turning numbers into percentages to size the bars
- Filtering a list based on what the user picks
- Keeping data saved with `localStorage`
